# LivingLab-Plugin User Guide

This user manual is designed to help you navigate the LivingLab-Plugin, which focuses on a comprehensive workflow feature that enables automatic, regular, and complete simulation in exchange with IDA-ICE and live sensor data. The plugin also provides individual workflow functions and data handling functions.

This user manual is intended to help you navigate the LivingLab-Plugin to enable well-functioning work with SIMULTAN.
